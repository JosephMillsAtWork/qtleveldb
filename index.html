QtLevelDB - Qt/QML wrapper for Google's LevelDB

This plugin provides a wrapper to use LevelDB with Qt in C++ or QML. See documentation for more information.

Installation

On Unix, just do: 

$ git clone https://github.com/paulovap/qtleveldb
$ qmake
$ make && make install

On Windows is very similar, just make sure you have Qt and QtCreator binaries in your path enviroment variable:

On Unix, just do: 

$ git clone https://github.com/paulovap/qtleveldb
$ qmake
$ jom && jom install
