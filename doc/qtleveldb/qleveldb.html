Detailed Description

The QLevelDB class enables access to a LevelDB database.

A QLevelDB classe can be used to insert or access data in a LevelDB database.

QLevelDBReadStream * QLevelDB::readStream(const QString startKey = QString(), const int length = -1)

Create a QLevelDBReadStream object to read a series of key/values. If no startKey or endKey is set it will stream all inserted elements. The order of the stream is sorted by key.
