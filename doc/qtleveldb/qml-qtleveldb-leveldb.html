Detailed Description

Property Documentation

batch : QLevelDBBatch*

Return batch object for bulk operations.


lastError : string

A string that indicates the status result of the last operation.


opened : bool

Indicate wheter the database is open or not.


options : QLevelDBOptions*

This property holds the configuration used to open the database. It cannot be changed once the database it opened


source : url

This property specifies the URL of datavase. The URL must point to a local file that contains LeveldDB's data.


status : LevelDB::Status

A enum that indicates the status result of the last operation.
