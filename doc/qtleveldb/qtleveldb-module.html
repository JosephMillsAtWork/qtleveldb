Qt LevelDB C++ Classes

List of C++ classes that provide the Qt LevelDB functionality. More...

This documentation was introduced in Qt 5.5.

Classes

QLevelDB

Enables access to a LevelDB database

QLevelDBBatch

Apply bulk operations in a LevelDB Database, rolling back in case of error

QLevelDBOptions

Manipulates options used when opening a LevelDB Database

QLevelDBReadStream

Provides a way to scan the database, receiving a stream of key/values in a ordered manner

Detailed Description

To use these classes in your application, use the following include statement:

#include <QtLevelDB/QtLevelDB>

To link against the module, add this line to your qmake .pro file:

QT += leveldb
